# SALARY NEGOTIATION — Offer Analysis + Negotiation Playbook

## Source Truth Anchors
- Negotiation playbook: `06_NEGOTIATION_PLAYBOOK.md` (JOBS-OS)
- Compensation database: `16_COMPENSATION_DB.md` (JOBS-OS)
- Decision framework: `29_DECISION_FRAMEWORK.md` (JOBS-OS)

## Protocol

### Step 1: Comp Analysis
- Check market range from Compensation DB for role type + level + location
- Cross-reference against company size, industry, funding stage
- Calculate: Low/Mid/High for role

### Step 2: Negotiation Targets
- **Stretch**: Top of range + 10% (ask first)
- **Target**: Midpoint of range (happy)
- **Walk-away floor**: Minimum acceptable (never share this)

### Step 3: Negotiation Strategy
1. Never disclose current/past salary. Redirect: "I'm focused on finding the right role where I can deliver maximum impact."
2. Never send pay slip.
3. Let them name the number first.
4. Use silence after their offer — let them fill it.
5. Negotiate in this order: Base → Bonus → Equity → Benefits → Start date

### Step 4: Total Compensation Breakdown
- Base salary negotiation script
- Bonus structure (sign-on, annual, performance)
- Equity (RSUs/options — value, vesting schedule)
- Benefits (extended health, dental, remote stipend, learning budget)
- Total comp comparison if multiple offers

### Step 5: Decision Matrix (If Multiple Offers)
- Compensation (total package)
- Role scope and growth
- Company stability and culture
- Remote flexibility
- PR sponsorship path (not mentioned in external comms, tracked internally)

### Step 6: Acceptance Script
- Thank them, express enthusiasm
- Confirm details in writing
- Ask about next steps
- Never accept same-day — ask for 24-48 hours

## Key Rules
- Never share salary history or pay slips
- Redirect any salary history question
- The right company respects you more for protecting your leverage
- Use FinOps section from 13-section package for company-specific targets
